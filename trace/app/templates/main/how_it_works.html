How Trace Works

Trace blends spaced repetition, active recall, and AI-assisted concept extraction so what you learn becomes usable expertise.

The Problem

The Ebbinghaus Forgetting Curve shows memory decays sharply without deliberate review. Passive consumption—highlighting, re-reading, bookmarking—doesn't create durable neural connections.

The Science

Spaced Repetition

Trace schedules concepts using SM-2 so you review right before you're about to forget, strengthening long-term retention efficiently.

Active Recall

You answer in your own words. Generating answers drives deeper encoding than recognition-based studying.

Testing Effect

Every recall attempt is a low-stakes test that boosts retention more than additional reading.

Import

Gemini concept extraction

Application reminders

Tie concepts to projects and receive prompts to apply them in real work. Learning becomes action.
